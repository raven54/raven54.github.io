프로토타입은 object와 link 2가지로 나눠서 이해해야 한다.

1. prototype object

  • 모든 객체의 조상은 ‘함수’이다.
  • 함수가 정의될 때는 2가지 일이 동시에 이루어진다. 첫번째는 해당 함수에 constructor(생성자) 자격이 부여되고, 두번째는 해당 함수의 prototype Object가 생성, 연결된다.
  • prototype Object는 constructor와 _ _proto__ 를 가지고 있고, 일반적인 객체와 같다.
