함수선언식 (Function Declarations)

함수선언은 Function Statement라고도 하며,
말 그대로 함수 문장이라는 뜻이다.
실행가능한 코드블럭이 아닌 함수의 정의를 나타내는 문장으로 해석되며 따라서 코드해것에 따른 수행결과가 존재하지 않는다는 것을 의미한다.

함수표현식 (Function Expresssions)

함수표현은 Function Literal이다. 실행 가능한 코드로 해석되거나 변수, 데이터구조에 할당되어지고 있음을 의미한다. 해당 코드블럭이 실행코드로서 해석되어지며 동시에 코드실행에 따른 결과값을 가지거나 특정 변수에 할당된 값으로 존재한ㄷ.ㅏ

위의 named function expression은 foo라는 변수에 이름있는 함수를 할당하고 있다.
이 named expression에는 특징이 있는데, 바로 해당 함수의 이름은 함수 밖에서 사용할 수 없다는 것이다.

호이스팅 : 선언문은 자바스크립트 엔진 구동시 가장 최우선으로 해석된다.

  • 선언문은 항상 최상위에서 작성하는 것을 권고하고 있다.
