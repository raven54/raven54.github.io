예제

메소드 호출 패턴

  • 함수를 객체의 속성에 저장하는 경우, 그 함수를 메소드라고 부른다.
  • 메소드를 호출할 때, this는 메소드를 포함하고 있는 객체 에 바인딩 된다.

예제

function add(a,b){
return a + b;
}

myObject.double = fuction(){
var that = this; // 대안
var helper = fuction(){
that.value = add(that.value, that.value);
}
helper();
};

myObject.double();
console.log(myObject.getValue()); // 6;
```
